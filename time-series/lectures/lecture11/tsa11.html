2 関数の作成

自作関数は次である.

VAR の残差に対するかばん検定

port.v <- function(x, lag) { # x = 推定結果, lag = ラグ
  res <- serial.test(x, lags.pt = lag, type = "PT.adjusted") # かばん検定
  res <- c(lag, res$serial$statistic, res$serial$parameter, res$serial$p.value) # ラグ, 検定統計量, 自由度, p 値

  return(res)
}
